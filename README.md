🧠 Multi-Language Quiz System

A console-based quiz application developed in C++ that allows users to test their knowledge through multiple-choice questions in different programming and web development subjects. The system provides a simple and interactive interface where users can choose a subject, answer randomized questions, and receive instant results with pass/fail status.

📌 Features
✅ Interactive console-based user interface
🌐 Multiple quiz categories:
C++
Java
HTML
🔀 Randomized questions for every attempt
🎯 Five unique questions per quiz session
📝 Student information collection (Name & Roll Number)
⚡ Instant feedback after each answer
📊 Automatic score and percentage calculation
🏆 Pass/Fail evaluation
🔁 Option to replay the quiz

🚀 How to Run
Using Code::Blocks / Dev-C++
Open the project file (Quiz.cpp).
Compile the program.
Run the executable.
Enter your:
Name
Roll Number
Select a subject.
Answer the questions by pressing the corresponding option key.
View your result and choose whether to play again.
